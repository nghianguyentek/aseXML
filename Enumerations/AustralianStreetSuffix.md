# AustralianStreetSuffix Enum

## Purpose

Define street suffixes as per Australian Standard AS4590:2006

### Version 4.0 additions

- `DE` - DEVIATION
- `ML` - MALL
- `IN` - INNER
- `OF` - OFF
- `ON` - ON
- `OP` - OVERPASS
- `OT` - OUTER

### Version 5.0 additions

Element AustralianStreetSuffix has been restricted to 2 characters
